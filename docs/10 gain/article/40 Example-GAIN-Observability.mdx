## One-line angle

AI observability is capture, retention, and replay — not another dashboard on token counts.

## Opening thesis

Teams instrument LLM calls and stop there. G.A.I.N Observability spans the **full loop**: what was retrieved, what was decided, what was executed, and what changed — so regulators, SREs, and architects can replay incidents months later.

---

## Why this domain needs G.A.I.N

Observability must follow G.A.I.N layers — not generic APM:

| Pillar | What you must see in production |
| --- | --- |
| **G** | Sources used · retrieval set · policy filter outcomes |
| **A** | Eval scores · drift · rollout cohort comparisons |
| **I** | Plan steps · tool calls · policy gate hits · escalations |
| **N** | Latency · cost · SLOs · multi-region health |

One Insight explains the observability gap. This article is the **telemetry architecture** for governed AI.

---

## Grounded (G)

- Log retrieval provenance: chunk IDs, source versions, ACL decisions
- Capture prompt assembly inputs — redacted per classification rules
- Store enough context to answer: *"Why did the system believe this was true?"*

---

## Adaptive (A)

- Quality metrics tied to eval harness versions — comparable over time
- Alert on distribution shift in retrieval or escalation rates
- Link production incidents to eval regression — not just error spikes

---

## Intelligent (I)

- Trace planner decisions as first-class spans — not buried in logs
- Policy gate outcomes: allow · deny · escalate — with rule ID
- Correlate agent tool chains end-to-end

---

## Native (N) — dominant

**Three capabilities enterprises actually need**

| Capability | Definition |
| --- | --- |
| **Capture** | Structured events at every G.A.I.N boundary — not raw text dumps |
| **Retention** | Tiered storage aligned to compliance (30d ops · 7y audit) |
| **Replay** | Reconstruct a request path for investigation — regulator-ready |

```text
Request → [G trace] → [I trace] → [N metrics] → Archive → Replay UI
```

**Patterns**

- OpenTelemetry-style traces across gateway · retrieval · model · tools
- Cost attribution per product line and tenant
- SLOs on P95 latency **and** policy-check latency

**Anti-patterns**

- Logging only final model output
- No retention policy until legal asks
