# 12 Weeks Plan

Ship consistent content to [jitendersharma.dev](https://jitendersharma.dev) while building **G.A.I.N** in public.

:::tip[Before you schedule]
Pick **2 Insights/week** from the [Content Bank](./Content-Bank) · tag each Insight → **[Tagging Guidelines for Insights](./Tagging-Guidelines-for-Insights)** · enrich assets → **[G.A.I.N](../10 gain/Types)** · **[Blueprints](../40 blueprint/Types)**
:::

## How this plan builds G.A.I.N

G.A.I.N is the operating model behind every row in the schedule — overlays in Phase 1, Decision Model in Phase 2, G.A.I.N Architecture + Maturity Model in Phase 3.

Full framework, overlay anchor weeks, and build track → **[G.A.I.N](../10 gain/Types)** · blueprint IP sequence → **[Blueprints](../40 blueprint/Types)**
