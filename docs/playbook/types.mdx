## Playbook vs Insight vs Blueprint

| Channel | Job | When to ship |
| --- | --- | --- |
| **Insight** | prove the argument | 2/week on cadence |
| **Blueprint** | visual reference model | after anchor Insight |
| **Playbook** | step-by-step implementation | after pattern is accepted |

```text
Insight  ->  Blueprint (optional)  ->  Playbook
```
