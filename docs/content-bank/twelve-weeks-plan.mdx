# 12 Weeks Plan

Ship consistent content to [jitendersharma.dev](https://jitendersharma.dev) while building **G.A.I.N** in public.

:::tip[Before you schedule]
Pick **2 Insights/week** from the [Content Bank](./Content-Bank) · tag each Insight → **[Tagging Guidelines](./tagging-guide/Tagging-Guidelines-for-Insights)** · enrich assets → **[G.A.I.N](../gain/Types)** · **[Blueprints](../blueprint/Types)**
:::

## Two tiers, one method

Every row is an **Insight**, but not the same size:

| Tier | Length | Where in the plan | Job |
| --- | --- | --- | --- |
| **Single-tension** (default) | ~1000–1500 words · 1 claim · routes depth down | most rows | provoke + reframe, link to depth |
| **Anchor** (flagship) | comprehensive, multi-diagram | **IP weeks 6 · 10 · 13** (🚩) | proof of authority |

:::important[Depth-first rule]
A single-tension Insight points to a deeper asset (G.A.I.N / Architecture / Playbook / Blueprint). **Draft or stub that sibling in the same week** — never publish an Insight that links to depth that does not exist. The IP weeks (6, 10, 13) are where the anchor Insight **and** its blueprint ship together.
:::

## How this plan builds G.A.I.N

G.A.I.N is the operating model behind every row — overlays in Phase 1, Decision Model in Phase 2, G.A.I.N Architecture + Maturity Model in Phase 3. Single-tension Insights **feed** the Signature Articles and blueprints that hold their depth.

Full framework, overlay anchor weeks, and build track → **[G.A.I.N](../gain/Types)** · blueprint IP sequence → **[Blueprints](../blueprint/Types)**

## Weekly workflow

```text
Monday     → Lock Tue + Thu · tier · Core pillar + voice + topic tags
Tuesday    → Draft + publish Insight · tag on site
Wednesday  → Draft + publish LinkedIn voice post for Tue Insight (same tags)
Thursday   → Draft + publish second Insight · tag on site
Friday     → LinkedIn post for Thu Insight · link related Insights · update Framework/Blueprint (anchor week)
```

**Pairing check each week:** every Insight published must have its **one LinkedIn voice post** with matching tags.

**Depth-first check each week:** every single-tension Insight must link to a depth sibling that is live or drafted this week.
