| Letter | Pillar | Covers |

| **G** | Grounded | Truth, context, knowledge alignment — RAG, retrieval, grounding, hallucination controls |
| **A** | Adaptive | Learning, feedback, evolution — eval, rollouts, feedback loops |
| **I** | Intelligent | Reasoning, agents, decision systems — orchestration, routing, boundaries |
| **N** | Native | Scalable, modular, future-ready — observability, ops, scale, compliance |

## Insight vs Signature Article

| | Insight | G.A.I.N Signature Article |
| --- | --- | --- |
| Job | prove one tension | map a whole domain |
| Length | ~1000–1500 words (single-tension) | comprehensive, multi-section |
| Tags | Core pillar + voice + topics | **none** (untagged) |
| Route | `/insights/*` | `/frameworks/gain-*` |

An Insight can be G.A.I.N-aligned. It is not a Signature Article until published on `/frameworks/gain-*`.
