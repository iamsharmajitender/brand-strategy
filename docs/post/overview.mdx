# Post Engine

The Post Engine is your template system for **LinkedIn voice posts** — **one per Insight**, same tags, shorter format.

:::info[Channel rule]
**Post Engine → LinkedIn.** **Insight Engine → Insights on site.** Never swap them.

**One Insight → one LinkedIn voice post.** Not a separate backlog. Not multiple posts per Insight.
:::

:::info[Tag every post — same as parent Insight]
**Tagging:** one Core pillar · one primary voice · one or more topic tags — **identical to the parent Insight** → [Tagging Guidelines](../content-bank/tagging-guide/Tagging-Guidelines-for-Insights) · [Core Pillars](../content-bank/tagging-guide/core-pillars) · [Voice Tags](../content-bank/tagging-guide/voice-tags) · [Topic Tags](../content-bank/tagging-guide/topic-tags). G.A.I.N → [G.A.I.N](../gain/Types).
:::

## Primary voice (type tags)

The LinkedIn post uses the **same voice** as its parent Insight → full catalog: [Voice Tags](../content-bank/tagging-guide/voice-tags).

## Workflow

```text
1. Insight published on site (Core pillar + voice + topic tags locked)
2. Open matching voice template (same voice as Insight)
3. Draft one short extract — one takeaway from THE CLAIM
4. Copy same tags from parent Insight
5. Link back to parent Insight on site
6. Publish on LinkedIn
```

## When to use each voice

Match the **Insight's primary voice** — do not pick a different voice for LinkedIn. Full catalog → [Voice Tags](../content-bank/tagging-guide/voice-tags#when-to-use-each-voice).
