## Architecture vs Blueprint vs Insight

| Channel | Job | Form |
| --- | --- | --- |
| **Insight** | prove one tension | ~1000–1500 words, tagged |
| **Architecture note** | durable systems reference | patterns + diagram + decisions |
| **Blueprint** | one-page visual model | layer table · flow · decision tree |

Notes may **inform** blueprints but are not duplicate long-form Insights.
